## Description
Lab Assignment 3 by Brandon Ortiz-Nguyen, Jeffery Trinh, Wenyang Li
This is a Python-based multiple-choice quiz program that presents users with 5 questions and evaluates their answers. The program tracks the user's score and saves it to a text file for future reference.

## Features
- 5 multiple-choice questions on various topics
- Input validation for answer choices
- Score tracking and display
- Automatic saving of results to "quiz_score.txt"
- User-friendly interface with clear feedback

## How to Run
1. Ensure you have Python 3.x installed on your system
2. Save the code as `quiz_program.py`
3. Open terminal/command prompt and navigate to the file directory
4. Run the program using: `python quiz_program.py`
5. Follow the on-screen instructions to answer the questions
